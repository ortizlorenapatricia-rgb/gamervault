Relaciones entre colecciones
categorias → juegos 1 categoría tiene N juegos. Referencia via juegos.categoriaId
plataformas → juegos N:M. Un juego tiene un array de plataformas. juegos.plataformas[]

usuarios → resenas 1 usuario tiene N reseñas. Máx 1 por juego. Via resenas.usuarioId
Pipelines de Agregación implementados

AGG 1 — Totales y promedios por categoría

$match$lookup$unwind $group$project$sort$limit
db.juegos.aggregate([ { $match: { precio: { $exists: true } } }, { $lookup: { from: "categorias", localField: "categoriaId", foreignField: "_id", as: "cat" } }, { $unwind: { path:"$cat", preserveNullAndEmptyArrays: true } }, { $group: { _id: "$cat.nombre", totalJuegos: { $sum: 1 }, totalVentas: { $sum: "$ventas" }, promedioMetacritic: { $avg: "$metacritic" }, precioPromedio: { $avg: "$precio" } } }, { $sort: { totalVentas: -1 } }, { $limit: 15 } ])

AGG 2 — Análisis por plataforma (JOIN + $unwind array)

$match$unwind$lookup $group$project$sort
db.juegos.aggregate([ { $match: { plataformas: { $exists:true,$ne:[] } } }, { $unwind: "$plataformas" }, { $lookup: { from: "plataformas", localField: "plataformas", foreignField: "_id", as: "plat" } }, { $unwind: { path:"$plat", preserveNullAndEmptyArrays: true } }, { $group: { _id: "$plat.nombre", totalJuegos: { $sum: 1 }, precioPromedio: { $avg: "$precio" }, descuentoPromedio: { $avg: "$descuento" } } }, { $sort: { totalJuegos: -1 } } ])

AGG 3 — Ranking Top N (configurable)

$match$lookup$unwind $project$sort$limit
db.juegos.aggregate([ { $match: { likes: { $gt: 0 } } }, { $lookup: { from: "categorias", localField: "categoriaId", foreignField: "_id", as: "cat" } }, { $unwind: { path:"$cat", preserveNullAndEmptyArrays: true } }, { $project: { nombre:1, desarrollador:1, likes:1, metacritic:1, ventas:1, calificacion:1, categoria: { $ifNull:["$cat.nombre","Sin cat."] }, score: "$likes" } }, { $sort: { score: -1 } }, { $limit: 10 } ])

AGG 4 — Análisis temporal por año

$match$group$project$sort
db.juegos.aggregate([ { $match: { fechaLanzamiento: { $exists: true, $type: "date" } } }, { $group: { _id: { $year: "$fechaLanzamiento" }, totalJuegos: { $sum: 1 }, promedioMetacritic: { $avg: "$metacritic" }, totalLikes: { $sum: "$likes" }, precioPromedio: { $avg: "$precio" }, totalVentas: { $sum: "$ventas" } } }, { $project: { _id: 0, anio: "$_id", totalJuegos: 1, promedioMetacritic: { $round: ["$promedioMetacritic", 1] } } }, { $sort: { anio: 1 } } ])

AGG 5 — Dashboard (estadísticas globales)

$group$projectcountDocuments
db.juegos.aggregate([ { $group: { _id: null, precioPromedio: { $avg: "$precio" }, precioMin: { $min: "$precio" }, precioMax: { $max: "$precio" }, totalLikes: { $sum: "$likes" }, totalVentas: { $sum: "$ventas" } } }, { $project: { _id: 0, precioPromedio: { $round: ["$precioPromedio", 2] }, precioMin: 1, precioMax: 1, totalLikes: 1, totalVentas: 1 } } ]) // + countDocuments en juegos, categorias, // plataformas, usuarios, resenas

$match$group$avg$sum
